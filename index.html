1. CSS Layout

Flexbox și Grid

Flexbox

Flexbox este folosit pentru alinierea elementelor pe o singură direcție: orizontal sau vertical. Este foarte util pentru meniuri, carduri și butoane.

Grid

Grid este folosit pentru layout-uri pe două direcții: rânduri și coloane. Este potrivit pentru pagini complexe și zone de conținut.

2. Bootstrap

Grid system și utilities

Grid System

Bootstrap folosește un sistem cu 12 coloane pentru layout-uri responsive. Clase precum container, row și col-md-4 ajută la organizarea conținutului.

container → row → col

Responsive Design

În Bootstrap, prefixele responsive precum sm, md, lg, xl și xxl nu se aplică doar la coloane. Ele pot fi folosite și pentru spacing, margini, padding, display, aliniere și alte clase utilitare.

sm ≥ 576px md ≥ 768px lg ≥ 992px xl ≥ 1200px xxl ≥ 1400px
col-md-4 | p-lg-5 | mt-sm-3 | d-md-flex

Utilities

Clasele utilitare Bootstrap ajută la stilizarea rapidă a elementelor, fără să fie nevoie să scriem CSS separat pentru fiecare detaliu.

d-flex gap-3 text-center p-3 m-2 mt-4 mb-4 justify-content-center align-items-center fw-bold rounded shadow

Componente Bootstrap utilizate

  • Navbar – meniul de navigare din partea de sus
  • Cards – fiecare secțiune este organizată în carduri
  • Buttons – butoane Bootstrap pentru interacțiune
  • Alerts – mesaje informative
  • Badges – etichete pentru componente
  • Grid System – layout responsive cu linii și coloane
  • Progress Bar – bară de progres
  • Accordion – componentă interactivă Bootstrap

Loading spinner

Loader-ul este făcut doar din CSS, cu border și animație keyframes.

animation: spin 1s linear infinite;

Exemplu de cod CSS

Mai jos este codul pentru animația de rotire folosită la loader.

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
